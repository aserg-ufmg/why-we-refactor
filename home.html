Why We Refactor? Confessions of GitHub Contributors - Dataset

This website contains the dataset from the following study:

Danilo Silva, Nikolaos Tsantalis, and Marco Tulio Valente, "Why We Refactor? Confessions of GitHub Contributors", 24th ACM SIGSOFT International Symposium on the Foundations of Software Engineering (FSE'2016), Seattle, WA, USA,November 13-18, 2016. [bibtex]

If you are interested to learn more about the process we followed for the creation of this dataset please refer to our paper. The tool we used to detect refactorings is available at GitHub:
https://github.com/tsantalis/RefactoringMiner
